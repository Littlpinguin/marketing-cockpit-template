## Vocabulary — Tier 2 (suspicious in clusters, 3+ = tell)

Robust, seamless, cutting-edge, innovative, comprehensive, pivotal, nuanced, compelling, transformative, bolster, underscore, evolving, fostering, imperative, intricate, overarching, unprecedented, vibrant, profound, renowned, stunning, showcasing, exemplifies, garner, valuable.

## Phrases and filler

- **"It's worth…" family**: "It's worth noting that…", "It's important to note…", "It bears mentioning…" → just state the thing.
- **"In today's…" family**: "In today's fast-paced world / digital landscape / competitive market", "In an era of…" → delete.
- **"At its core…" / "At the end of the day…" / "When all is said and done…"** → delete or restructure.
- **Hedging starters**: "While there are certainly…" (pick a side), "To be sure…" (delete), "That said…" ("But", once max), "This is not without its challenges…" (name the challenge), "It remains to be seen…" (delete).
- **AI closers**: "The bottom line is…", "Only time will tell", "The future is bright", "Start your journey today", "Exciting times lie ahead" → delete or be specific.
- **Filler**: "In order to" → "To" ; "Due to the fact that" → "Because" ; "At this point in time" → "Now" ; "In the event that" → "If" ; "Has the ability to" → "Can".
- **Chatbot artifacts**: "Great question!", "Certainly!", "I hope this helps!", "Let me know if you'd like me to expand", "You're absolutely right!" → delete entirely.
- **Knowledge-cutoff disclaimers**: "While specific details are limited…", "Based on available information…" → source it or delete. ("As of [date]" is normal in data journalism.)

## Significance inflation

| Pattern | Fix |
|-|-|
| "marking a pivotal moment in the evolution of" | State what happened, not its cosmic significance |
| "stands/serves as a testament to" | Delete; state the evidence |
| "underscores/highlights its importance" | Delete; importance should be self-evident |
| "reflects broader trends" | Name the trend or delete |
| "setting the stage for" | Say what actually happened next |
| "represents a shift" / "key turning point" | Be specific about what changed |
| "indelible mark" / "deeply rooted" | Delete or be specific |

## Grammar-level patterns

- **Copula avoidance** (tell = clustering): serves as / stands as / represents / marks → "is" ; boasts / features / offers → "has".
- **Superficial -ing phrases**: "highlighting…", "underscoring…", "emphasizing…", "reflecting…", "symbolizing…", "showcasing…", "contributing to…", "fostering…", "encompassing…" → delete or expand into a sourced sentence.
- **Negative parallelisms**: "Not only… but also…", "It's not just about X, it's about Y" — dans ce template : faute fatale dès la première occurrence (checklist §2c).
- **Rule of three**: flag only when the third item is padding or a near-synonym.
- **Synonym cycling**: "protagonist… main character… central figure… hero" → pick one term.
- **False ranges**: "From X to Y" with no meaningful scale → just list the topics.

## Structure

- **Formulaic section**: Setup → Explanation → "What this means for you" — vary endings; some sections just stop.
- **Parallel list**: every bullet same structure and length — let some be short, some full sentences.
- **Tricolon close**: "Start building, start learning, start growing." → one direct statement.
- **Balanced conclusion**: good news / challenge / inspiring closer → end on one strong statement.
- **Em dash**: dans ce template, zéro cadratin (`—`) — règle de marque absolue (checklist §2b), plus stricte que la source.

## Tone

- **Too balanced** — real experts pick a position and defend it.
- **Too positive** — "exciting developments", "tremendous opportunity" ; real writing is measured, occasionally pessimistic.
- **Too smooth** — real writing has slight roughness: an aside, a pivot, a blunt statement after a complex one.
- **No personality** — no opinions, no edge. (Injecter la voix de `01-brand/voice.md`, pas une personnalité générique.)

## Detection heuristic (5+ hits = likely AI)

- [ ] 3+ Tier 1 words
- [ ] 5+ Tier 2 words
- [ ] "It's worth noting" or similar filler present
- [ ] Every section follows the same structure
- [ ] "Not X, but Y" present (une seule occurrence suffit ici)
- [ ] Em dashes present (zéro toléré ici)
- [ ] No sentences under 8 words
- [ ] No informal language anywhere
- [ ] Perfectly balanced conclusion
- [ ] Every list has the same number of items
- [ ] No opinions, asides, or personality
- [ ] Hedging on every claim
- [ ] Copula avoidance multiple times
- [ ] Significance inflation in opening or closing
- [ ] Chatbot artifacts present
